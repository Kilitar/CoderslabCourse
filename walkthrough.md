# Data Science Course - Welcome! 🚀

Tento projekt byl vytvořen vaším AI parťákem pro účely vašeho rekvalifikačního kurzu.

## Struktura projektu
- /data: Zde ukládejte své CSV, Excel nebo jiné datasety.
- /notebooks: Pro vaše Jupyter Notebooky (.ipynb).
- /scripts: Pro čisté Python skripty (.py).
- /models: Pro ukládání natrénovaných modelů.
- /.venv: Vaše oddělené Python prostředí.

## Jak začít
1. Otevřete tuto složku ve svém editoru.
2. Aktivujte virtuální prostředí (v PowerShellu: .\.venv\Scripts\Activate.ps1).
3. Zkuste spustit svůj první skript: python scripts\hello_data.py.

## Nainstalované knihovny
- **Pandas**: Král tabulek a manipulace s daty.
- **NumPy**: Základ pro numerické výpočty.
- **Matplotlib & Seaborn**: Pro tvorbu grafů a vizualizací.
- **Scikit-learn**: Váš nástroj pro Machine Learning.

Hodně štěstí v kurzu! Kdykoliv budete potřebovat něco naprogramovat, spočítat nebo vysvětlit, jsem tady.
