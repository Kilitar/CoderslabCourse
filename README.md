# New Python Project

This is a fresh start for a Python project in the `Data-Science-Course` workspace.

## Getting Started

1. Create a virtual environment:
   ```bash
   python -m venv .venv
   ```
2. Activate the virtual environment.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the project:
   ```bash
   python main.py
   ```
